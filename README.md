# Backend API

This is the backend server built using Node.js, Express.js, and MongoDB.  
It handles authentication, API requests, database operations, and email services.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (Authentication)
- Nodemailer

## Setup

### Install Dependencies


```bash
npm install  // to install all the file that are needed

PORT=
MONGODB_URL=
CORS_ORIGIN=
JWT_SECRET=
REFRESH_TOKEN_SECRET=
BACKEND_URL=
FRONTEND_URL=
EMAIL_USER=
EMAIL_APP_PASSWORD=

npm run dev //Run the Server
